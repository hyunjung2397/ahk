##key
ctrl + F7 : Start
ctrl + F8 : Pause On/Off
ctrl + F9 : Exit

##setup.ini
[searchoption]
variation : 이미지 검색시 픽셀 색상의 정확도. 숫자가 0이면 정확히 동일한 색에 대해서만 반응. 숫자가 255이면 모든 색에 대해 반응. 0~255.
transparent : 이미지에서 검색시 제외할 색상을 지정한다. 000000 ~ FFFFFF 또는 White와 같이 문자로 지정할 수 있다.  
interval : 각 이미지 간 검색 간격
wait : 이미지 검색 성공 후 대기시간
ringfile : 이미지 검색 성공 시 재생되는 소리 파일
imgpath : 검색을 원하는 이미지 파일 경로

[monitoroption] : 이미지 검색 대상 모니터. 값이 1이면 사용, 0이면 사용안함
